This manual is designed to provide additional detailed explanations of the features in the user guide.

---

## SignUp page
To access the applications, users must sign up or sign in through email and password.

Users can:
- Create an account if they don't have one
- Sign In

---

## Dashboard

After logging in, dashboard consists of these features:
- Workouts
- Recipes
- Favorites
- Community
- Mesaage Icon
- Chatbot

---

## Workout Feature

Includes all workouts that are organized by muscle group and stored in the database.

Users can:
- Filter workouts through muscle groups
- Search by keyword
- Save to favorites

Each exercise consists of:
- Name
- Target muscle group
- Ideal sets and repetitions
- Tutorial
- Muscle movement animation

---

## Recipes

Includes all recipes in the database system.

Users can:
- Filter by tag
- Save to favorites

Each recipe consists of:
- Ingredients
- Nutritional information
- Preparation steps
- Tags (e.g., High Protein, Vegan)
- Ocassionally a video/ link to an external site


---

## Favorites

Includes all the favorited exercises and recipes

Users can:
 - Access workouts and recipes easily
 - Delete it

---

## Messaging

A real time messaging system

Users can:
- Add friends
- Text friends

---

## AI Chatbot

The chatbot retrieves accurate data from the database.

Users can ask for:
- Workout planning questions
- Nutrition advice
- General fitness guidance


---

## Account Settings

Users can:
- Update profile information
- Change password
- Delete account
