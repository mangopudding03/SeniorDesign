## Milestones List


1. **User Research**
    - Conduct and complete user interviews to understand goals and needs of users.  
    - **Deliverable:** Final user research findings and insights.

2. **Web Page Features**
    - Implement a homepage layout  
    - Implement a navigation bar for food and exercises  
    - Integrate BMI body outline tool API  
    - **Deliverable:** A navigation bar and a BMI body outline tool on the homepage.

3. **Curated Exercise Tutorials**
    - Gather exercise tutorials  
    - Categorize them by muscle groups and fitness goals  
    - **Deliverable:** A curated exercise tutorial database.

4. **Food Database**
    - Identify and integrate a reliable food and nutrition database API  
    - Store food items with macro and calorie data for quick lookup  
    - **Deliverable:** A well-populated food database containing nutritional data.

5. **Macro Tracking Tool**
    - Build macro tracking functionality  
    - Connect frontend to backend to retrieve nutritional values  
    - **Deliverable:** Functional macro tracker tool.

6. **Testing and Final Deliverable**
    - Test all components including:  
        - Navigation bar  
        - BMI body outline visualization  
        - Food database  
        - Exercise tutorials  
        - Macro tracking  
    - **Deliverable:** A fully tested web application.
